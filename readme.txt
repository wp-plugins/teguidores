== Description ==

Este plugin muestra de forma visual tus seguidores de Twitter, facebook y tus lectores de FeedBurner con sus tres botones correspondientes.

-----------------------------------

In English:

This plugin show your Twitter Followers, Facebook Fans and FeedBurner readers with three social icons.



== Installation ==

1. Subir el archivo 'teguidores.zip' al directorio '/wp-content/plugins/' donde tienes alojado tu blog.
2. Descomprimir el archivo 'teguidores.zip'
3. Activar el plugin a través del menú 'Plugins' de Wordpress.
4. Modifica el archivo Visual.php  del plugin.
5. ¡Usa el widget!
  

-----------------------------------

In English:

1. Upload the file 'teguidores.zip' to your '/wp-content/plugins/' directory when you host your blog.
2. Unzip the 'teguidores.zip' file.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Modify the visual.php archive
5. ¡Use the widget!
