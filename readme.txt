Teguidores es un plugin que te permite mostrar de forma visual tu número de seguidores en Twitter, Facebook y RSS.

-----------------------------------

In English: Teguidores is a plugin show your Twitter followers, Facebook Fans and Feedburner Readers.
== Description ==

Este plugin muestra de forma visual tus seguidores de Twitter, facebook y tus lectores de FeedBurner con sus tres botones correspondientes.

-----------------------------------

In English:

This plugin show your Twitter Followers, Facebook Fans and FeedBurner readers with three social icons.

== Installation ==

1. Subir el archivo 'teguidores.zip' al directorio '/wp-content/plugins/' donde tienes alojado tu blog.
2. Descomprimir el archivo 'teguidores.zip'
3. Activar el plugin a través del menú 'Plugins' de Wordpress.
4. Modifica el archivo Visual.php  del plugin.
  

-----------------------------------

In English:

1. Upload the file 'teguidores.zip' to your '/wp-content/plugins/' directory when you host your blog.
2. Unzip the 'teguidores.zip' file.
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Modifique the visual.php archive

== Frequently Asked Questions ==



= Como modifico el archivo visual.php =

el archivo visual.php viene comentado, pero si no eres capaz de modificarlo correctamente, aquí te dejo las instrucciones:
	1. En la línea 7 modifica Tu-usuario-de-twitter por tu usuario de Twitter.
	2. En la línea 14 modifica "Tu-ID-de-facebook" por tu el id de tu página de facebook.
	3. En la lína 21 modifica "Feedburner" por tu usuario de Feedburner.

-----------------------------------

In English:

= How to modifique the visual.php archive =

visual.php is commented, but if you don know how to modify the archive, follow the instructions:
	1. In line 7 change "Tu-usuario-de-Twitter" for your Twitter Username.
	2. In line 14 change "Tu-ID-de-Facebook" for your facebook fan page ID
	3. In line 21 change "Feedburner" for your feedburner username.
